# Typography

The Typography component provides various heading styles and body text. Three Google fonts make up our typography, each of which are referenced below

## Headings

Hero headings use the font type [Black Han Sans](https://fonts.google.com/specimen/Black+Han+Sans). The remaining headings use [Anek Latin](https://fonts.google.com/specimen/Anek+Latin?query=Anek+Latin).

<Canvas>
  <Story name="Headings" argTypes={{ hero: { control: 'text' }, headingTwo: { control: 'text' }, headingTwoAlt: { control: 'text' }, headingThree: { control: 'text' }, headingFour: { control: 'text' } }}>
    {({ hero, headingTwo, headingTwoAlt, headingThree, headingFour }) => (
      <VStack align="left">
        <Heading as="h1" variant="h1">
          {hero || 'Hero/Heading 1 - H1'}
        </Heading>

        <Heading as="h2" variant="h2Alt">
          {headingTwo || 'Heading 2 - H2 alternative'}
        </Heading>

        <Heading as="h2" variant="h2">
          {headingTwoAlt || 'Heading 2 - H2'}
        </Heading>

        <Heading as="h3" variant="h3">
          {headingThree || 'Heading 3 - H3'}
        </Heading>

        <Heading as="h4" variant="h4">
          {headingFour || 'Heading 4 - H4'}
        </Heading>

        <Heading as="h5" variant="h5">
          {headingFour || 'Heading 5 - H5'}
        </Heading>

        <Heading as="h6" variant="h5">
          {headingFour || 'Heading 6 - H6'}
        </Heading>
      </VStack>
    )}

  </Story>
</Canvas>

## Body

The Body component contains the main body text. Body text uses [Roboto Slab](https://fonts.google.com/specimen/Roboto+Slab?query=Roboto+).

<Canvas>
  <Story
    name="Body"
    argTypes={{
      bodyDefault: { control: 'text' },
      bodyLarge: { control: 'text' },
    }}
  >
    {({ bodyDefault, bodyLarge }) => (
      <VStack align="left">
        <Text>{bodyDefault || 'Body/Default'}</Text>
        <Text fontSize="2xs">{bodyLarge || 'Body/Large'}</Text>
      </VStack>
    )}
  </Story>
</Canvas>

## Subtitles

Subtitles use [Anek Latin](https://fonts.google.com/specimen/Anek+Latin?query=Anek+Latin)

<Canvas>
  <Story
    name="Subtitles"
    argTypes={{
      subtitleDefault: { control: 'text' },
      subtitleBold: { control: 'text' },
      subtitleLarge: { control: 'text' },
    }}
  >
    {({ subtitleDefault, subtitleBold, subtitleLarge }) => (
      <VStack align="left" fontFamily="bodyAlternative">
        <Text fontSize="sm">{subtitleDefault || 'Subtitle/Default'}</Text>
        <Text as="b" fontSize="sm">
          {subtitleBold || 'Subtitle/Bold'}
        </Text>
        <Text as="b" fontSize="xl">
          {subtitleLarge || 'Subtitle/Large'}
        </Text>
      </VStack>
    )}
  </Story>
</Canvas>
