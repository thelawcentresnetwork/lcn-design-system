# Typography

The Typography component provides various heading styles and body text. Three Google fonts make up our typography, each of which are referenced below

## Headings

Hero headings use the font type [Black Han Sans](https://fonts.google.com/specimen/Black+Han+Sans). The remaining headings use [Anek Latin](https://fonts.google.com/specimen/Anek+Latin?query=Anek+Latin).

    h2Conference2024: { control: 'text'},
    h3Conference2024: { control: 'text'},

    {({ hero, headingTwo, headingTwoAlt, headingThree, headingFour, h1Conference2024, h2Conference2024, h3Conference2024, h4Conference2024}) => (

  </Story>
</Canvas>

## Body

The Body component contains the main body text. Body text uses [Roboto Slab](https://fonts.google.com/specimen/Roboto+Slab?query=Roboto+).

<Canvas>
  <Story
    name="Body"
    argTypes={{
      bodyDefault: { control: 'text' },
      bodyLarge: { control: 'text' },
      bodyDefaultConference2024: { control: 'text' },
      bodyLargeConference2024: { control: 'text' },
    }}
  >
    {({ bodyDefault, bodyLarge, bodyDefaultConference2024, bodyLargeConference2024 }) => (
      <VStack align="left">
        <Text>{bodyDefault || 'Body/Default'}</Text>
        <Text fontSize="2xs">{bodyLarge || 'Body/Large'}</Text>
        <Text fontSize="bodyConference2024" fontFamily="bodyConference2024">{bodyDefault || 'Body / Default / Conference 2024'}</Text>
        <Text fontSize="bigBodyConference2024" fontFamily="bodyConference2024">{bodyLarge || 'Body / Large / Conference 2024'}</Text>
      </VStack>
    )}
  </Story>
</Canvas>

## Subtitles

Subtitles use [Anek Latin](https://fonts.google.com/specimen/Anek+Latin?query=Anek+Latin)

<Canvas>

      bigSubtitleConference2024: { control: 'text' },

    {({ subtitleDefault, subtitleBold, subtitleLarge, smallSubtitleConference2024, bigSubtitleConference2024 }) => (
