# Page hero

The Hero component is used to establish visually appealing sections on webpages. It consists of a main heading, subheading, and an optional small label at the top of the heading. On home pages the Hero component can also include an image.

## Overview

These can be used on all pages excluding a HomePage. Properties of this component include: `backgroundColour`, `textColour`, `heading`, `pageTitle` and a `subHeading`
